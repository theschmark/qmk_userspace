Keymap for TGR MIA with the followign fun features:

1. Vim mode
2. Windows & Mac order for alt/win auto-inferred.
    - FN + target windows key to chagne the settings
    - FN + top right OSL -> target windows key to save as default in eeprom
3. Dual shifts cause all caps
4. Hold tab brings navigation mode
5. Right side macros. Top right vim off, macro1, macro2, fn, right
    - FN + top right OSL -> target macro to start record
    - FN + top right OSL -> top right to stop
